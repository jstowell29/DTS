Message System

The Evo messaging system allows for multiple types of messages Text, Voice, and Fax. Each message can also have an attachment of different media types. This information is described in the getMessageList (detail) below. This guide will focus primarily on returning the number of “new” messages. Listing a user’s inbox, reading messages and deleting messages. It will also go into detail on how to send a message.

Number Of New Messages

MessageBox.getMessageList (Summary Mode)

This service will return the number of new messages. The count will be located in the field “NEW”. This is the hit that we use to return the number of new messages. The hit returns numbers on different kinds of messages, but we only need worry about the “new” field.

Here is an example call:

/get http://dts.myvoffice.com/atomic/dts/index.cfm?jsessionid=84305142d986e79841c8e7551b5775713a14&service=MessageBox.getMessageList&apikey=*****&distid=1004&DETAIL=No&SELECTION=All&format=JSON

Details on the input and output parameters for MessageBox.getMessageList can be found here.
