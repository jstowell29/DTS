How to enroll a member with an initial order and autoship

This guide will walk you through how to enroll a new person, place an order and create an autoship. This process assumes you are using protect pay as your payment processor.

Step 1: Session.login

First we will grab a session. This is done using session.login. A session is essential to make any subsequent calls.

Here is an example call:

/get http://dts.myvoffice.com/clientname/dts/index.cfm?service=Session.login&apikey=***apikey_here&PASSWORD=***password_here***&USERID=***username_here***&format=JSON

Output will look like this

{
	  "ROLES": "distributor",
	  "SESSION": "8430b5a9399b52628798706c427278511a43"
	}
