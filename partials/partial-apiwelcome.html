DTS API Documentation

Welcome to the DTS API Documentation. Listed are all service modules available through DTS. For each service you will find an inclusive list of input and output fields associated with each service.

The documentation is organized into Modules, which are listed on the menu on the left. The modules include Autoship, CommissionHub, Distributor, DTParameters, Earnings, FastTreeAdmin, Genealogy, Group, MessageBox, Orders, Points, ProgressTab, ProtectPay, Qualifications, ReportGroups, Reports, Session, Telephony, Evo, WebMeeting.

All of these services are available to be used throughout your application. However, each Module has its own set of intricacies which this documentatino will describe.

Modules

Autoship

This handler is associated with autoship templates. Transactional based services are located under the Orders handler. This handler includes a service for returning autoship template information.

CommissionHub

The commission hub handler includes services to integrate your ERP system and use the InfoTrax commission engine to payout commissions.

Distributor

Everything related to members within your system. These services include member information, enrollment, personal information, suspend, and terminating.

DTParameters

All services related to DataTrax parameters. Use these services to return member statuses, bonus types, rank information, and next numbers for orders.

Earnings

Services return commission payout information. Earnings, paid on reports and detail reports.

FtAdmin

Services for FastTree file information. Use these services to find details of fields available within FastTree and file import dates and times.

Genealogy

These services are used to run genealogy reports using FastTree.

Group

Business group services which allow you to manage a users differen business groups. A great communication tool for users and their downline. Manage current members and prospects.

MessageBox

Unleash the power of the InfoTrax messagining platform. Real time communication for your corporate or your distributor base. No longer do you need to deal with mass email for communication.

Orders

An entire suite of transactional based services. Create one-time and auto-delivery orders. Retrieve historical order information.

ProgressTab

An entire set of services for training modules. Manage the steps you want your member base to take to achieve success and build their business.

ProtectPay

Fully integrated ProtectPay service for retrieving payment information on file with ProtectPay.

Qualifications

Up to the second commission information to know how close a member is to achieving the next rank. Built for creating InfoTrax's robust rank qualification module.

ReportGroups

Organize your FastTree reports in groups for powerful comparison information.

Reports

Coming soon...The new reporting module from Infotrax. Create organizational and transactional reporting based on the InfoTrax patented FastTree technology.

RunReports

Part of the reporting module. These are the services used to run your reports that have been created by you or by InfoTrax.

Session

Session creation for authentication for using DTS and its many services.

Telephony

VoiceCast services. Create and manage your toll free number.

Evo

Evo is InfoTrax's renowned member facing backoffice. Create and manage member Evo account information.

Webmeeting

WebMeeting is InfoTrax's platform for streaming teleconferences. Record now... broadcast whenever. Services avalable iclude listing WebMeetings published by users. More on there way...
